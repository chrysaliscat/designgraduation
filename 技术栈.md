# 🎓 农产品销售平台 - 真实技术栈清单 (2026 毕设版)

本项目以企业级前后端分离开源框架 RuoYi-Vue 为底座，深度结合农产品垂直电商业务进行构建。以下是本项目在当前开发环境中的**真实运行版本**。

---

## 1. 核心运行环境 (Runtime Environment)
| 组件 | 实际运行版本 | 说明 |
| :--- | :--- | :--- |
| **Java (JDK)** | **1.8.0_481** | 开发与运行的核心基石 |
| **Node.js** | **v22.21.1** | 前端构建与开发服务器运行时 |
| **npm** | **11.5.2** | 前端依赖包管理工具 |
| **Maven** | **3.9.12** | 后端项目构建与依赖管理 |
| **MySQL** | **8.0.44** | 关系型数据库存储引擎 |
| **Redis** | **5.0.14** | 高性能 Key-Value 缓存中间件 |

---

## 2. 后端核心架构 (Backend Stack)
*   **服务端主框架**：`Spring Boot 2.5.15` (长期维护版本，稳定性极佳)
*   **安全与鉴权**：`Spring Security 5.7.14` + `JWT` (实现 RBAC 权限控制)
*   **持久层框架**：`MyBatis 3.5.x` (结合 `PageHelper 1.4.7` 物理分页)
*   **连接池组件**：`Druid 1.2.27` (带监控功能的高性能连接池)
*   **JSON 解析**：`Fastjson2 2.0.60` (高性能安全性解析库)
*   **API 文档**：`Swagger / Springfox 3.0.0` (在线接口自动化文档)

---

## 3. 前端交互架构 (Frontend Stack)
*   **视图层框架**：`Vue.js 2.6.12` (渐进式 JavaScript 框架)
*   **UI 组件库**：`Element UI 2.15.14` (桌面端后台管理组件库)
*   **状态管理**：`Vuex 3.6.0` (全局数据状态共享)
*   **路由管理**：`Vue Router 3.4.9` (SPA 单页面路由控制)
*   **网络请求**：`Axios 0.28.1` (基于 Promise 的 HTTP 客户端)
*   **可视化图表**：`ECharts 5.4.0` (用于销量统计与数据大屏)

---

## 🏗️ 架构特点 (Architecture Highlights)
1.  **前后端分离**：Web 端与后端 API 服务器完全解耦，通过 JWT 令牌进行通信。
2.  **热重启支持**：已配置 Maven 与 Webpack 热更新，代码修改后秒级生效，极大提升毕设开发效率。
3.  **标准化部署**：兼容 Windows 本地开发环境及 Linux Docker 自动化部署。
